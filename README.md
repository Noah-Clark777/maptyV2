Map My Fitness – Activity Tracking App

A fitness tracking application that allows users to log workouts, monitor activity, and visualize performance data through an interactive and responsive interface.
This project focuses on building a data-driven application that handles user input, tracks activity metrics, and updates the UI dynamically.

Features:
* Track workouts (running, walking, cycling, etc.)
* Map-based activity tracking
* Display key metrics (distance, duration, pace)
* Log and manage workout sessions
* Real-time UI updates based on activity data
* Fully responsive design

Tech Stack:
* Frontend: JavaScript (ES6+), HTML5, CSS3
* Architecture: MVC / modular JavaScript
* Maps: Leaflet
* Data Handling: Local state/browser storage
* Deployment: Netlify

Architecture & Key Concepts:

* Geolocation API:
Uses browser geolocation to capture and display the user's position.

* Map Integration:
Renders workouts visually using an interactive map interface.

* State Management:
Maintains a structured state for workouts and user activity.

* Object-Oriented Programming (OOP):
Model workouts (running, cycling) as objects with shared and unique properties.

* Event-Driven UI:
Updates the interface based on user actions like adding or selecting workouts.

* Persistent Data (if implemented):
Stores workouts in local storage to maintain state across sessions.
